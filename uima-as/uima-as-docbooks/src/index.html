Documentation for UIMA Asynchronous Scaleout (UIMA-AS)

UIMA-AS is built on top of the base UIMA implementation. The distributed binary packaging for UIMA-AS includes the base UIMA framework, but does not include the documentation. You can find the documentation for base UIMA on the main UIMA documentation website, http://incubator.apache.org/uima/documentation

The UIMA-AS documentation is available in both PDF and HTML formats.
